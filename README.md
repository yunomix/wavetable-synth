# Wavetable Synth

A 6-voice polyphonic wavetable synthesizer using Web Audio API AudioWorklet and TypeScript.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the dev server:
   ```bash
   npm run dev
   ```

## Features
- **6-Voice Polyphony**
- **Custom Wavetable Drawing**
- **ADSR Envelope Control**
- **MIDI File Playback**
